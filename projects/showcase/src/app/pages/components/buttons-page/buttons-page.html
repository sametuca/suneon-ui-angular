Buttons

Collection of interactive neon-themed button components with stunning animations.

Magnetic Button

Interactive button that magnetically pulls towards the cursor.

Small Primary Large
Secondary Outline
Non-Magnetic
HTML
<!-- With magnetic effect (default) -->
<sn-magnetic-button variant="primary" [magnetic]="true">
  Magnetic Button
</sn-magnetic-button>

<!-- Without magnetic effect -->
<sn-magnetic-button variant="primary" [magnetic]="false">
  Normal Button
</sn-magnetic-button>

Properties

Name Type Default Description
magnetic boolean true Enable/disable magnetic effect
variant 'primary' | 'secondary' | 'outline' 'primary' The visual style of the button
size 'sm' | 'md' | 'lg' 'md' The size of the button
strength number 0.3 Magnetic pull strength (0-1, only when magnetic=true)

Shimmer Button

Button with a continuous animated neon shimmer effect.

Default Shimmer Neon Shimmer Rainbow Shimmer
HTML
<sn-shimmer-button variant="neon">
  Neon Glow
</sn-shimmer-button>

Properties

Name Type Default Description
variant 'default' | 'neon' | 'rainbow' 'default' The color theme of the shimmer effect
size 'sm' | 'md' | 'lg' 'md' The size of the button
disabled boolean false Disables the button
