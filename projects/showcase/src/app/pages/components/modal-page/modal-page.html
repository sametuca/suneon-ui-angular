NeonModal Component

Create stunning modal dialogs with neon effects, smooth animations, and multiple color variants.

Modal Variants

This is a primary modal dialog. It displays important information and actions with a cyan neon theme. Click the buttons below to interact with the modal.

Operation completed successfully! This modal uses a green neon theme to indicate positive results.

This action cannot be undone. Are you sure you want to proceed? This modal uses a red neon theme to indicate caution.

Properties

Property Type Default Description
isOpen boolean false Controls modal visibility
title string '' Modal header title
variant string 'primary' Color variant: primary, danger, success
showFooter boolean true Show/hide action buttons
onClose EventEmitter - Emitted when modal is closed
onConfirmed EventEmitter - Emitted when confirm button is clicked

Example Usage

<sn-modal
  [isOpen]="isModalOpen"
  title="Confirm Delete"
  variant="danger"
  [showFooter]="true"
  (onClose)="closeModal()"
  (onConfirmed)="handleDelete()"
>
  <p>Are you sure you want to delete this item?</p>
  <p>This action cannot be undone.</p>
</sn-modal>
