API Reference

Inputs

Property Type Default Description
items DropdownItem[] [] Array of dropdown options
value any null Currently selected value (two-way binding)
placeholder string 'Select an option' Placeholder text when nothing is selected
variant 'primary' | 'secondary' | 'success' | 'warning' | 'danger' 'primary' Color variant
size 'sm' | 'md' | 'lg' 'md' Size variant
disabled boolean false Disables the dropdown
openDirection 'down' | 'up' 'down' Direction the menu opens

Outputs

Property Type Description
valueChange EventEmitter<any> Emitted when value changes (for two-way binding)
selectionChange EventEmitter<DropdownItem> Emitted with the full item object when selection changes

DropdownItem Interface

Property Type Description
label string Display text for the option
value any Value associated with the option
icon string Optional icon (emoji or text)
disabled boolean Whether the option is disabled
divider boolean Renders a divider instead of an option
