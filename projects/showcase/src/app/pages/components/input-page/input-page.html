NeonInput Component

Create elegant, neon-styled form inputs with smooth focus animations. Support for labels, placeholders, multiple input types, and color variants.

Color Variants

Properties

Property Type Default Description
label string '' Label text displayed above input
placeholder string '' Placeholder text inside input
type string 'text' Input type: text, password, email, number, etc.
variant string 'primary' Color variant: primary, secondary, success, danger
size string 'md' Input size: sm, md, lg
disabled boolean false Disable input field
value string '' Input value (supports two-way binding with [(value)])

Input Types

Example Usage

<sn-neon-input
  type="email"
  label="Email"
  placeholder="your@email.com"
  variant="primary"
  size="md"
  
  (onChange)="onEmailChange($event)"
></sn-neon-input>

Size Properties

Use the size property to control input dimensions:

<sn-neon-input size="sm"></sn-neon-input>
<sn-neon-input size="md"></sn-neon-input>
<sn-neon-input size="lg"></sn-neon-input>
