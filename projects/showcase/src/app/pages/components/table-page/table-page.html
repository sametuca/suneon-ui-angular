NeonTable Component

Display data in elegant, neon-styled tables with scrolling, striping, and hover effects.

Table Variants

{{ variant | titlecase }} Variant

ID Name Email Status {{ row.id }} {{ row.name }} {{ row.email }} {{ row.status }}

Properties

Property Type Default Description
variant string 'primary' Color variant: primary, secondary, dark
striped boolean false Enable striped rows for better readability
hover boolean false Enable hover effect on rows
scrollable boolean false Enable horizontal/vertical scrolling

Example Usage

<sn-table 
  variant="primary" 
  [striped]="true" 
  [hover]="true"
  [scrollable]="true">
  <thead>
    <tr>
      <th>Name</th>
      <th>Email</th>
      <th>Status</th>
    </tr>
  </thead>
  <tbody>
    <tr *ngFor="let row of tableData">
      <td>{{ '{' }}{{ '{' }} row.name {{ '}' }}{{ '}' }}</td>
      <td>{{ '{' }}{{ '{' }} row.email {{ '}' }}{{ '}' }}</td>
      <td>{{ '{' }}{{ '{' }} row.status {{ '}' }}{{ '}' }}</td>
    </tr>
  </tbody>
</sn-table>
