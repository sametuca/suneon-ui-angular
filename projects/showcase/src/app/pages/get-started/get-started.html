Getting Started

Installation

This guide will help you get started with Suneon's neon components in just a few minutes.

Prerequisites

Before you can install and use Suneon, make sure your project meets the following requirements:

  • Angular: Version 19 or higher.
  • TypeScript: Version 5.7 or higher.

Install Components

1

Initialize an Angular Project

Suneon is fully compatible with Angular standalone components. To get started, set up your Angular project:

Terminal 
ng new my-app --standalone
cd my-app

If you already have a project compatible with Angular 19+, you can skip this step.

2

Install Suneon

Install the Suneon package via npm:

Terminal 
npm install suneon
3
